# danil-semenov
Зарегистрируйтесь на сайте [exchangerate-api.com](https://exchangerate-api.com).
Подтвердите почту и зайдите в Dashboard.
Скопируйте свой уникальный API Key.
Вставьте его в коде `main.py` в переменную `self.API_KEY`.


Запустите программу: `python main.py`.
Введите сумму (число).
Выберите валюты и нажмите "Конвертировать".
История последних 20 запросов сохранится в файл `history.json`.
